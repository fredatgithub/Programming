1. What is the purpose of the "Hello World!" program?
  It shows us the basic setup of a program in almost every language. This also shows us if our compiler is working.

2. Name the four parts of a function.
  Return type, name, parameters, and body.

3. Name a function that must appear in every C++ program.
  main

4. In the "Hello, World!" program, what is the purpose of returning 0?
  This shows success if the program finished.

5. What is the purpose of the compiler?
  The compiler changes the code you wrote into code that the CPU can understand called binary.

6. What is the purpose of the #include directive?
  This tells the compiler to make the files specified available to this file.

7. What does a .h suffix at the end of a file name signify in C++?
  The file is a header file

8. What does the linker do for your program?
  Links all the .o files into a single executable

9. What is the difference between a source file and an object file?
  The source file is more readable to humans, and the object file is more readable to the CPU and stuff.

10. What is an IDE and what does it do for you?
  An IDE is a program that makes programming easier, it usually helps compile and link your programs together.

11. If you understand everything in the textbook, why is it important to practice?
  Because practicing would help make you understand it more.
